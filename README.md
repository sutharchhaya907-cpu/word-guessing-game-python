# Word Guessing Game (Python)
<br>

##  Description
This is a console-based Word Guessing Game developed using Python. The game randomly selects a secret word based on difficulty level, and the player must guess it within a limited number of attempts. Hints are provided after each incorrect guess, and the game maintains a history of attempts using file handling.
<br>

##  Features
- Difficulty levels: Easy, Medium, Hard
- Attempt limit for each game
- Hint generation using string comparison
- Game history storage using file handling
- Option to view and clear history
- Modular code using functions
- <br>

## Technologies Used
- Python
- File Handling
- Functions
- Loops and Conditional Statements
- String Manipulation
